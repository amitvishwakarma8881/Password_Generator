# Password Generator

A secure and customizable **password generator** that helps you create strong, random passwords to protect your online accounts.

## Features
- Generate passwords of any length
- Include or exclude:
  - Uppercase letters
  - Numbers
  - Special characters
- User-friendly interface
- Lightweight and fast
